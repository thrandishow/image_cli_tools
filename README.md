# Image CLI Tools

**A powerful and easy-to-use command-line tool for batch image processing.**

</div>

---

`Image CLI Tools` provides a simple interface to perform common actions on your images directly from the terminal. Whether you need to resize, convert, or compress a single image or a whole directory, this tool has you covered.

## âœ¨ Features

- **Resize:** Change image dimensions by percentage or specific width/height.
- **Convert:** Convert images between formats (e.g., `PNG` to `JPEG`).
- **Compress:** Reduce file size with adjustable quality settings.
- TODO: **Rotate:** Rotate images by a given angle.
- **Batch Processing:** Apply actions to multiple files in a directory at once.

## Requirements

To use this tool, you'll need the following on your system:

- **Python 3.14+**

## Installation

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/your-username/image_cli_tools.git
    cd image_cli_tools
    ```

2.  **Create and activate a virtual environment:**

    ```bash
    python3 -m venv .venv
    source .venv/bin/activate
    ```

3.  **Install the required packages:**
    ```bash
    pip install -r requirements.txt
    ```
